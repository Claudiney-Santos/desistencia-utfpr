# desistencia-utfpr
Projeto 2 da disciplina de Inteligência Artificial.

# Descrição
Dada a base de dados parcial de alunos da UTFPR, o objetivo do projeto é criar um modelo preditivo que identifique quais alunos têm maior probabilidade de desistir do curso.

# Dados
Para o treinamento, é recomendado o uso do arquivo `all_in_balanceado.csv`.

Os campos campos abaixo indicam a quantidade de matérias em que o aluno teve determinado status:
- `Aprovado por Nota/Frequência`;
- `Reprovado por Nota/Frequência`;
- `Crédito Consignado`;
- `Reprovado por Nota`;
- `Aprovado em Exame de Suficiência`;
- `Aprovado por Nota`;
- `Reprovado por Frequência`;
- `Reprovado em Exame de Suficência`;
- `Reprovado`;
- `Aprovado por Frequência`;
